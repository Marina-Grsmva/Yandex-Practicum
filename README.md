# Анализ рынка заведений общественного питания Москвы

# Цель исследования
Проанализировать рынок заведений общественного питания Москвы и выявить потенциально прибыльное направление с учетом локации, целевой аудитории и среднего чека.

# Исходные данные
Датасет с заведениями общественного питания Москвы, составленный на основе данных сервисов Яндекс Карты и Яндекс Бизнес в период с июня по август 2022 года.

# Используемые библиотеки Python
*Pandas, numpy, re, matplotlib, seaborn, plotly.express, plotly, plotly.subplots, folium, folium.plugins*
